## Sprint Goal:
Continue adding features to ChromaNotes to make the user experience even better

## Sprint 3 User Stories:
- US10: As a user, I want to customize the background theme such as making it light, dark, or pastel so that the app matches my mood.
- US11: As a user, I want to be able to insert images into my notes so that I am able to visually support my ideas or incorporate reference diagrams
- US12: As a user, I want to be able to set reminders and get notified about my due dates so that I don't forget important tasks
- US13: As a user, I want to search for notes by title, content or tags so that I can find them quickly
- US14: As a user, I want to be able to archive or hide older notes so that my workspace remains uncluttered
- US15: As a user, I want to export my notes as PDF or text files so that I can back them up or share them easily
