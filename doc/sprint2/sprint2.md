## Sprint Goal:
Implement the ability to add tags to notes, sort them into folders, colour code them, pin them, and write them in different fonts and styles

## Team Capacity: 30 Hours

## Sprint 2 User Stories:
- US05: As a user, I want to be able to organize my notes into folders or categories so I can keep related ideas together (Not Completed)
- US06: As a user, I want to be able to assign tags to notes, so I can group them by topic
- US07: As a user, I want to pin my important notes so that they always appear at the top of my workspace
- US08: As a user, I want to colour-code my notes so that I can easily identify different topics and priorities
- US09: As a user, I want to write in different font styles and sizes so that my notes are more readable and fit my preferences
- US10: As a user, I want to filter notes by date created or last modified, so that I can locate my recent work easily
